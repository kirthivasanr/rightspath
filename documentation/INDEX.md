## 🚀 Quick Links

### To Start the Project:
1. Backend: Run `start-backend.bat` from the root directory
2. Frontend: Run `start-frontend.bat` from the root directory

### Tech Stack:
- **Frontend**: React + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Python + Flask + ChromaDB + Sentence Transformers
- **AI**: OpenRouter API (LLaMA model)
- **Auth**: Firebase Authentication
